Project Title: AI Blood Donation & Availability System

Project Goal:
Build an AI-powered platform that manages blood donor registration, real-time blood availability across hospitals, and emergency blood request matching.

Main Features:
- Donor registration with blood group and location
- Hospital blood stock dashboard
- Real-time blood availability tracking
- Emergency blood request system
- AI-based donor and hospital matching
- AI shortage prediction
- AI chatbot for blood requests

System Type:
Web application with dashboard + AI features

Actors:
- Donor
- Hospital / Blood Bank
- Admin
- User / Patient

Core Modules:
1. Donor Registry
2. Blood Stock Management
3. Emergency Request System
4. AI Matching System
5. AI Prediction System
6. Dashboard UI

Rules for AI tools:
- Generate clean UI using modern dashboard design
- Use simple database structure
- Prefer fast hackathon-friendly stack
- Keep components modular
- Use REST API structure
- Use real-time updates if possible

AI Features to implement:
- Suggest best donor based on blood group and location
- Predict blood shortage using current stock
- Respond to emergency request using AI
- Provide chatbot for quick search

Priority:
1. Working demo
2. Clean UI
3. AI feature visible
4. Fast response system
